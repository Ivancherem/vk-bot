# AI Nexus VK Бот

🤖 Продвинутый бот для сообщества ВКонтакте "Заработок на AI-сервисах"

## Команды бота

| Команда | Описание |
|---------|----------|
| привет | Приветствие |
| промокоды | Все актуальные промокоды |
| партнерки | Партнёрские ссылки |
| сайт | Ссылка на ai-toolkit.ru |
| идеи | Способы заработка |
| инструменты | Популярные AI |
| обучение | Ресурсы для изучения |
| факт | Интересный факт |
| шутка | Пошутить |
| мотивация | Поднять настроение |
| статистика | Ваша активность |
| помощь | Список команд |

## Развертывание на Render

1. Форкните этот репозиторий
2. Создайте Background Worker на Render
3. Добавьте переменную окружения `VK_TOKEN`
4. Готово!
